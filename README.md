## Requirement

`tailwindcss` ^v1.9.0

## Installation

```
yarn add innovasive-ui-react
```

Update `tailwind.config.js`

```js
module.exports = {
  presets: [require('innovasive-ui-react/src/tailwind.config')],
}
```

Add styles to your `main.css`

```css
@import 'innovasive-ui-react/src/styles/index.css';
```
